# Capstone

Multi-discipline project spanning hardware, software, and ML systems.

## Structure

| Directory | Purpose |
|-----------|---------|
| `hardware/` | Schematics, CAD, BOM, datasheets, firmware, test logs |
| `software/` | Application code, tests, config |
| `ml/` | Datasets, experiments, models, training, evaluation |
| `docs/` | Cross-cutting documentation and decision log |
| `tools/` | Build scripts, Docker, CI helpers |

## Quick Start

```bash
just --list        # See available tasks
```
